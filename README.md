# vidgen

A small practice project containing a FizzBuzz implementation and pytest unit tests.

## Files

- `fizzbuzz.py` — Prints FizzBuzz from 1 to `n`.
- `tests/test_fizzbuzz.py` — Pytest unit tests for the FizzBuzz function.
- `skills/` — Pi agent skill files for Python coding and testing.

## Running tests

```bash
python -m pytest -q
```

## Requirements

- Python 3.14+
- pytest
